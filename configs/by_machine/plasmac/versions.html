v0.090 - 2020 ene 09
  • agregar actualizaciones automáticas de configuración de PlasmaC
  • cambiar el nombre de algunos pines y variables para mantener la coherencia

El configurador se actualizará a esta versión automáticamente la próxima vez que se inicie LinuxCNC.
Si no desea actualizaciones automáticas, agregue MANUAL_UPGRADE = 1 a la sección [PLASMAC] del archivo .ini.

Hay una serie de cambios de nombre de pin HAL. Si tiene una configuración estándar de PlasmaC, el configurador se encargará de los cambios de nombre.

Si tiene alguno de estos en los comandos HAL personalizados o en un archivo HAL personalizado, deberá cambiarles el nombre para adaptarlo.
Nuevo nombre Viejo nombre
inicio de manchado centro-spot-on
umbral de manchado umbral de punto central
tiempo de observación tiempo central
inicio de escritura escribano
escriba-en-retraso escriba-inicio-retraso
brazo de escriba brazo de escriba de aire
escribano Air-Scribe-Start
inicio de corte husillo
