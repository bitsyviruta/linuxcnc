corexy_by_kins.ini

Simulación CoreXY usando el
módulo de cinemática corexykins

$ man kins para más información

Notas:

1) Usar corexykins requiere
interruptores home que funcionen en posiciones
de motor para las articulaciones 0 y 1

2) Un método alternativo para la
cinemática coreXY utiliza cinemática trivkins
con un componente hal:
     (corexy_by_hal.comp)
Este método permite el uso de interruptores
home alineados a movimiento coordinado x,y.

Vea la demostración: corexy_by_hal.ini
