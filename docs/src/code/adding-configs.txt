= Agregando elementos al selector de configuraciones

Se pueden agregar Configuraciones de ejemplo al selector de configuración
mediante dos métodos:

* Aplicaciones auxiliares - Aplicaciones instaladas independientemente
  con un paquete deb puede colocar subdirectorios de configuración en
  un directorio especificado del sistema. El nombre del directorio se especifica
  usando el script de shell linuxcnc_var:

====
  $ linuxcnc_var LINUXCNC_AUX_EXAMPLES
  /usr/share/linuxcnc/aux_examples
====
* Configuración de tiempo de ejecución -- el selector de configuraciones también puede ofrecer
  subdirectorios de configuración especificados en tiempo de ejecución utilizando una
  variable de ambiente exportada (LINUXCNC_AUX_CONFIGS). Esta
  variable debe ser una lista de rutas de una o más directorios de configuraciones
  separados por (:). Normalmente, esta variable
  se establecería en un shell que inicia linuxcnc o en un script de inicio de usuario
  ~/.profile. Ejemplo:

====
  export LINUXCNC_AUX_CONFIGS=~/myconfigs:/opt/otherconfigs
====
