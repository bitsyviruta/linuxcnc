= Agregar elementos de selección de configuración

Se pueden agregar configuraciones de ejemplo al Selector de configuración
por dos métodos:

* Aplicaciones auxiliares: aplicaciones instaladas independientemente
  con un paquete deb puede colocar subdirectorios de configuración en
  Un directorio del sistema especificado. Se especifica el nombre del directorio.
  utilizando el script de shell linuxcnc_var:

====
  $ linuxcnc_var LINUXCNC_AUX_EXAMPLES
  / usr / share / linuxcnc / aux_examples
====

* Configuración de tiempo de ejecución: el selector de configuración también puede ofrecer
  subdirectorios de configuración especificados en tiempo de ejecución utilizando un
  variable ambiental exportada (LINUXCNC_AUX_CONFIGS). Esta
  La variable debe ser una lista de rutas de una o más configuraciones
  directorios separados por un (:). Típicamente, esta variable
  se establecería en un shell iniciando linuxcnc o en un usuario
  ~ / .profile script de inicio. Ejemplo:

====
  exportar LINUXCNC_AUX_CONFIGS = ~ / myconfigs: / opt / otherconfigs
====
