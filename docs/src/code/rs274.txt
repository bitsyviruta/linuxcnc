= Interprete independiente

El intérprete independiente rs274 está disponible para su uso a través de la línea de comando.

== Uso

----
Uso: rs274 [-p interp.so] [-t tool.tbl] [-v var-file.var] [-n 0|1|2]
          [-b] [-s] [-g] [input file [output file]]

    -p: especifica el intérprete conectable a usar
    -t: especifica el archivo .tbl (tabla de herramientas) a usar
    -v: especifica el archivo .var (parámetros) a usar
    -n: especifica el modo:
           0: continuar
           1: entrar en modo MDI
           2: parada (predeterminado)
    -b: Aternar la bandera 'block delete' (por defecto: OFF)
    -s: Alternar la bandera 'print stack' (valor predeterminado: OFF)
    -g: Alternar la bandera 'go (modo por lotes)' (valor predeterminado: OFF)
    -i: especificar el archivo .ini (predeterminado: sin archivo ini)
    -T: llamada a task_init()
    -l: especificar log_level (por defecto: -1)
----

== Ejemplo

Para ver el resultado de un bucle, por ejemplo, podemos ejecutar rs274 en el siguiente archivo
y ver que el bucle nunca termina. Para salir del bucle, usar Ctrl-Z.
Se necesitan los siguientes dos archivos para ejecutar el ejemplo.

.test.ngc
----
#<test> = 123.352

o101 while [[#<test> MOD 60 ] NE 0]
(debug,#<test>)
    #<test> = [#<test> + 1]
o101 endwhile

M2
----

.test.tbl
----
T1 P1 Z0.511 D0.125 ;1/8 end mill
T2 P2 Z0.1 D0.0625 ;1/16 end mill
T3 P3 Z1.273 D0.201 ;#7 tap drill
----

.comando
----
rs274 -g test.ngc -t test.tbl
----
